# Inline_Highlight_ObsidianPlugin
Plugin for Obsidian

**Allows you to highlight individual phrases inline using very simple syntax.**
You can configure text color, highlight background, and some other parameters in the plugin settings.

**Examples:**
```
- If a question ?? arises while studying a note ? 
- Then I can mark it as something !! that needs attention !  
- And even set up additional :: highlighting :
```

I'm not even a hobbyist programmer — I just wanted to leave this kind of comment + highlight and couldn't find any suitable plugins.  
I asked ChatGPT to write a plugin for me. I’ll refine it and release it as open source.

I want the plugin to be flexible and convenient to use for any preference, but the core value of its functionality for me is maximum speed and ease of inline use. 
I don't plan to turn it into HTML callouts.

Your help and suggestions are welcome!

The current release is quite usable. Known issues:
- Highlighting breaks if there are multiple comments on the same line
- It's not possible to highlight more than one line
